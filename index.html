Getting Started

Welcome to the Market Assistant Platform (MAP) documentation. This guide will help you get started with the platform.

Prerequisites

  • Node.js 18+
  • npm 9+ or yarn 1.22+
  • PostgreSQL 14+

Installation

# Clone the repository
git clone https://github.com/Linced/MAP.git
cd MAP

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local

# Start the development server
npm run dev

Project Structure

Frontend

Next.js 14+ application with TypeScript and Tailwind CSS

apps/web/

API

Backend services and API endpoints

apps/api/

Packages

Shared libraries and utilities

packages/

Development

Available scripts for development:

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start

# Run tests
npm test

# Run linter
npm run lint

# Format code
npm run format
